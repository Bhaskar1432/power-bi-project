# 📱 Mobile Sales Dashboard – Power BI Project

Hi 👋  
This is my Power BI project where I analyzed mobile sales data from an Excel file and created an interactive dashboard to generate meaningful business insights.

---

## 📌 Problem Statement

The Excel file contained raw mobile sales data, but it was:

- Unstructured
- Difficult to analyze
- Hard to identify trends manually
- Time-consuming to generate reports

The goal of this project was to:

✔ Clean and transform the Excel data  
✔ Build a data model  
✔ Create an interactive Power BI dashboard  
✔ Generate clear business insights  

---

## 🛠 Tools & Technologies Used

- 📊 Power BI
- 📄 Microsoft Excel (Data Source)
- 📈 Data Visualization
- 🧮 DAX (Data Analysis Expressions)
- 🔄 Data Cleaning & Transformation (Power Query)

---

## 📂 Project Files

## 📊 Dashboard Features

- 📌 Total Sales KPI
- 📌 Total Quantity Sold
- 📌 Profit Analysis
- 📌 Sales by Brand
- 📌 Sales by Model
- 📌 Monthly Sales Trends
- 📌 Region-wise Performance
- 📌 Interactive Filters & Slicers
